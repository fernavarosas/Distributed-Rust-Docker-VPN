# Proyecto 1 - VPN + Docker + Rust Distribuido

## Objetivo

Implementar un entorno distribuido con 5 máquinas virtuales Ubuntu Server sin GUI, conectadas mediante ZeroTier CLI, con Docker en cada nodo y ejecución de un programa distribuido en Rust bajo modelo coordinador-workers.
