## Contingencia aplicada en `coord-01`

Debido a una incidencia persistente en `worker-01`, se desplegaron dos contenedores worker adicionales en `coord-01` para continuar las pruebas distribuidas del sistema.

Esta medida no reemplaza la arquitectura planeada del proyecto, pero permitió validar la comunicación y ejecución coordinador-workers sin detener el avance técnico del sprint.
